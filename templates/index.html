Purpose

This site is to be used as a support for the Japanese B classes. The contents are broken up as below.

  • The Language: This page holds a discription of the language.
  • Course Structure: This page shows a basic layout of what we will cover over the 15 weeks.
  • Resources: This is a storage
  • Test: This will allow you to test yourself against the vocabulary for each week, or allow you to upload your own .csv files to create your own tests.
  • Charts: You will find the Hiragana and Katakana charts here.
  • Kana Test: works like test, but for Katakana. You should be able to select the line you want to test. Just like we do in class.
  • Kanji Test: as above, but for Kanji
  • Links: This page contains links to other sites
