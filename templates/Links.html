Jisho

This is a relatively user friendly Japanese/English online dictionary, which is also one of the few to allow students to search Japanese terms in Romaji. It doesn't match the ability of other on-line dictionaries that cater for translators. However, it is perfect for the beginner learner.

Amounst other things, this website supports:
  • Japanese -> Enlish lookup
  • English -> Japanese lookup
  • Kanji Lookup via radicals
  • Kanji Lookup via kanji sketching
  • Example Sentences

Quizlet

This is a free drilling website that allows users to create user defined flash card sets, which can then be shared public with others. This website also supports both android and iphone apps that also allow students to learn while out and about and on the move.

Amounst other things, this website supports:
  • JFlash card creation
  • flash card drillingp
  • the ability to search other user created flash card sets
  • Games that drill the flash card pairs
  • Audio support
